# 呉の避難所一覧

呉市の避難所情報を確認できるWebアプリケーションです。CSVファイルをアップロードして、避難所情報を検索・閲覧できます。

## 🌍 機能

- **避難所検索**: 名前や住所で避難所を検索
- **エリアフィルター**: エリア別に避難所を絞り込み
- **詳細表示**: 避難所の住所、電話番号、収容可能人数などを表示
- **レスポンシブデザイン**: スマートフォンやタブレットにも対応
- **赤色ベースデザイン**: 緊急性を示す赤色ベースのUI

## 📋 CSVファイル形式

避難所データは以下の形式のCSVファイルで提供してください：

```
避難所名,住所,エリア,電話番号,収容可能人数,備考
中央小学校,呉市中央1-1-1,中央区,082-1234-5678,500,体育館利用可
東小学校,呉市東1-2-3,東区,082-2345-6789,300,教室利用可
```

### フィールド説明

| フィールド名 | 説明 | 必須 |
|-----------|------|------|
| 避難所名 | 避難所の名前 | ○ |
| 住所 | 避難所の所在地 | ○ |
| エリア | 避難所が属するエリア/区 | ○ |
| 電話番号 | 連絡先電話番号 | ○ |
| 収容可能人数 | 収容できる人数（数字） | △ |
| 備考 | その他の情報 | △ |

## 🚀 使い方

### ブラウザで開く

```bash
# CSVファイルのアップロード画面が表示されます
# ファイルを選択ボタンから避難所データのCSVファイルを選択してアップロード
```

### ファイルのアップロード後

1. **避難所一覧表示**: アップロード後、避難所が一覧で表示されます
2. **検索**: 避難所名や住所で検索
3. **エリア絞り込み**: ドロップダウンメニューからエリアを選択
4. **詳細確認**: カードをクリックして避難所の詳細情報を確認

## 🎨 デザイン特性

- **赤色ベースUI**: 緊急時を示す赤色グラデーション
- **カードレイアウト**: 避難所情報をカード形式で表示
- **モバイル対応**: レスポンシブデザインでスマートフォンにも対応
- **検索機能**: リアルタイム検索で素早く避難所を探検可能
