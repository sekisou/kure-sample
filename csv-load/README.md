# ジェネリック医薬品ダッシュボード

## 概要
先発医薬品と後発医薬品のデータを可視化するダッシュボードアプリケーションです。
HTMLとJavaScriptのみで構築されており、サーバーやフレームワークは不要です。

## 機能

### データアップロード
- CSVファイルをブラウザから直接アップロード
- ファイル読み込み後、自動的にダッシュボードを表示

### サマリーカード
- 先発品種類数
- 後発品種類数
- 総数量
- 平均薬価差

### グラフ表示
1. **後発品医薬品 使用数量トップ10** - 棒グラフ
2. **先発品と後発品の薬価比較** - 比較棒グラフ
3. **後発品医薬品別 数量分布** - 棒グラフ

### 詳細テーブル
- 全データを表形式で表示
- 検索機能
- ページネーション
- 表示件数変更（10/25/50/100件）

## CSVフォーマット

```csv
年月,先発医薬品名,先発医薬品薬価,後発医薬品名,後発医薬品薬価,数量,単位
2024-10,薬品A,100.5,ジェネリックA,50.2,1000,錠
```

## 使用方法

1. `index.html` をブラウザで開く
2. 「CSVファイルを選択」ボタンをクリック
3. CSVファイルを選択
4. 自動的にダッシュボードが表示されます

## 技術スタック

- HTML5
- CSS3
- JavaScript (Vanilla)
- Chart.js (グラフ描画ライブラリ)
